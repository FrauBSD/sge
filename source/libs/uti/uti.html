Overview

Libuti is a low level library with several utility functions. This library bases on functionality of librmon.


Comparing Hosts

There is one function in libuti which is worth being mentioned. The function hostcmp() is used in nearly all cases when two hostnames are compared. This is done under consideration of the setting of 'default_domain' and 'ignore_fqdn' in the cluster configuration (see sge_conf(5)).
